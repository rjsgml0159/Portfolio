## Overview
This mini project explores user experience and perceived usability issues in a campus-related mobile application. The study focuses on how users interpret interaction flow, friction points, and overall ease of use.

## Objectives
- Design a mock UX research framework
- Identify potential usability and interaction issues
- Analyze user perception and response patterns
- Translate observations into design implications

## Methods
A simulated UX research workflow was developed, including survey design and structured evaluation of user experience factors. Responses were examined to detect common pain points, behavioral tendencies, and perceived usability challenges.

## Tools & Approaches
- UX Research Principles
- Survey Design
- Qualitative / Perception-Based Analysis
- User Experience Evaluation
